# TSX 기능 테스트

이 파일은 MDX에서 TSX 기능이 제대로 작동하는지 테스트하기 위한 것입니다.

## 마크다운 기능

- **굵은 글씨**와 *기울임체* 사용 가능
- [링크](https://example.com) 사용 가능
- `인라인 코드` 사용 가능

## 코드 블록

```javascript
function hello() {
  console.log("Hello, MDX!");
  return <div>React 컴포넌트도 가능!</div>;
}
```

## TSX 컴포넌트 테스트

아래는 React 컴포넌트를 사용한 예시입니다:

```tsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  
  return (
    <div>
      <p>현재 카운트: {count}</p>
      <button onClick={() => setCount(count + 1)}>
        증가
      </button>
    </div>
  );
}
```
