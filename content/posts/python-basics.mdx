# Python 기초 다지기

Python은 가독성이 좋고 배우기 쉬운 프로그래밍 언어입니다.

## 기본 문법

### 변수와 데이터 타입
```python
# 숫자
age = 25
height = 175.5

# 문자열
name = "홍길동"
message = '안녕하세요'

# 불린
is_student = True
is_working = False

# 리스트
fruits = ["사과", "바나나", "오렌지"]
```

### 함수 정의
```python
def greet(name):
    return f"안녕하세요, {name}님!"

# 함수 호출
result = greet("철수")
print(result)  # 출력: 안녕하세요, 철수님!
```

Python의 기본 문법을 마스터하면 다양한 프로젝트를 시작할 수 있습니다!
