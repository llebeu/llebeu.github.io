# 웹 접근성 가이드

웹 접근성은 모든 사용자가 웹사이트를 사용할 수 있도록 하는 중요한 요소입니다.

## 기본 원칙

### 1. 인식 가능성 (Perceivable)

정보와 사용자 인터페이스 요소는 사용자가 인식할 수 있어야 합니다.

```html
<!-- 좋은 예시 -->
<img src="chart.png" alt="2024년 매출 현황 차트" />

<!-- 나쁜 예시 -->
<img src="chart.png" />
```

### 2. 조작 가능성 (Operable)

사용자 인터페이스 요소와 네비게이션은 조작 가능해야 합니다.

```html
<!-- 키보드로 접근 가능한 버튼 -->
<button type="button" onclick="submitForm()">
  제출하기
</button>
```

### 3. 이해 가능성 (Understandable)

정보와 사용자 인터페이스 조작은 이해할 수 있어야 합니다.

웹 접근성을 향상시키면 더 많은 사용자가 웹사이트를 효과적으로 사용할 수 있습니다!
